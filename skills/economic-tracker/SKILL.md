# economic-tracker

## Description

经济效益追踪器 - 记录和分析项目收益、时间成本、ROI等经济指标，帮助量化AI辅助创作的价值。支持项目收益记录、时间追踪、成本分析、ROI计算等功能。

## When to Use

当用户需要以下经济相关操作时激活此技能：
- 记录项目收入或支出
- 追踪项目时间投入
- 计算项目ROI
- 查看收益报表
- 分析项目盈利能力
- 对比不同项目的经济效益

## Tools

此技能需要使用以下工具：
- `Read` - 读取经济数据文件
- `Write` - 创建和更新经济记录
- `SearchReplace` - 修改现有记录
- `Glob` - 查找经济相关文件

## Economic Metrics

### 基础指标
- **总收入** - 所有项目收入总和
- **总支出** - 所有项目支出总和
- **净利润** - 总收入 - 总支出
- **总时间** - 所有项目时间投入（小时）

### 效率指标
- **时薪** - 净利润 / 总时间
- **项目平均收益** - 总收入 / 项目数
- **时间利用率** - 计费时间 / 总时间

### 项目指标
- **项目ROI** - (项目收入 - 项目支出) / 项目支出 × 100%
- **项目时薪** - 项目净利润 / 项目时间
- **项目利润率** - 项目净利润 / 项目收入 × 100%

## Report Types

### 日报
- 今日收入/支出

### 周报

- 项目进度更新

### 月报
- 月度收益总结
- ROI分析
- 项目对比
- 趋势预测

### 项目报告
- 项目完整经济分析
- 投入产出比
- 盈利能力评估

## Best Practices

1. **及时记录** - 收入和支出发生后立即记录
2. **详细描述** - 记录清晰的描述便于后续分析
3. **准确计时** - 时间记录要准确，区分计费和非计费时间
4. **定期复盘** - 每周/每月查看报表，优化时间分配
5. **目标设定** - 设定收入目标和时间预算

## Examples

### 示例1：完整项目追踪

用户：我接了一个技术文档代写的项目，报价2000元，预计需要10小时

执行：
1. 创建项目 "技术文档代写-客户A"
2. 记录预期收入 2000元
3. 开始计时
4. 工作完成后停止计时
5. 生成项目报告

### 示例2：月度收益分析

用户：帮我分析一下这个月的收益情况

执行：
1. 读取本月所有交易记录
2. 计算总收入、总支出、净利润
3. 统计时间投入
4. 计算时薪和ROI
5. 生成月度报表

### 示例3：项目对比

用户：对比一下内容创作和软件开发的收益情况

执行：
1. 按类别汇总项目数据
2. 计算各类别的平均时薪
3. 对比ROI和利润率
4. 生成对比分析报告
